This is the API

addComment

By calling this operation you can search for movies in the system.

URL

http://moviesapi.codecat.pl/comments

HTTP Method

POST

Response Type

List

Parameters

  • Query: Movie String

    pass an optional movie id to get comments only for specific movie

addMovie

Adds movies to database

URL

http://moviesapi.codecat.pl/movies

HTTP Method

POST

Response Type

Parameters

  • Body: Title MovieQuery(MovieQuery)

    Movie title to add - more informations will be fetched from OMDB API.

findMovies

By calling this operation you can search for movies in the system.

URL

http://moviesapi.codecat.pl/movies

HTTP Method

GET

Response Type

List

Parameters

  • Query: id String

    Pass an optional movie id for looking up for specific movie.

  • Query: search String

    This param can be used for searching. All words will be tested against movie title.

  • Query: limit Integer

    maximum number of records to return
